🎮 プロジェクト概要

LoL Item Quizは、League of Legendsのアイテム知識を効率的に学習するためのWebアプリケーションです。ゲーム感覚で楽しみながら、アイテムの関係性や価格を覚えることができます。

✨ 主な機能

⚔️

Component Master

基本コンポーネントから完成アイテムを推測するクイズモード。アイテムビルドの理解を深めます。

💰

Price Master

アイテムの正確な価格を当てるクイズモード。ゴールド効率の最適化に役立ちます。

🧮

Number Wizard

数学的思考力を鍛える計算チャレンジ。ゲーム中の瞬時の判断力向上をサポートします。

🛠️ 技術スタック

🐍 Python Flask
🎨 HTML/CSS/JS
🌐 Riot Games API
💾 Session Storage

📊 データソース

本アプリケーションはRiot Games公式のData Dragon APIを使用しており、常に最新のパッチ情報に対応しています。現在はPatch {{ patch_version }}に対応しています。
